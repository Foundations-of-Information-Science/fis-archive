Dear Luis,

My mathematical work on music theory
has shown in detail that musical works are
maximal memory stores. In fact, this is part of
a general argument I have given that
art-works are maximal memory stores,
which was developed in my book
"A Generative Theory of Shape"
(Springer-Verlag, 550pages).

If you want to read something on
this music theory, there is a paper
on the FIS web-site, at the following
location:

Musical Works are Maximal Memory Stores.
http://fis.iguw.tuwien.ac.at/resources/papers/Leyton_Music_Paper.pdf
